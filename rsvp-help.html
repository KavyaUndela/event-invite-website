Step-by-Step Setup

1 Go to Firebase Console

Open Firebase Console and log in with your Google account.

2 Create Firestore Database

  • Click Firestore Database from left menu
  • Click Create Database
  • Choose Production mode
  • Select your region
  • Click Create

3 Update Firestore Security Rules

Go to Firestore Database ‚Üí Rules tab and paste:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /rsvps/{document=**} { allow read, write: if true; } match /events/{document=**} { allow read, write: if true; } } }

Click Publish

4 Enable Authentication

  • Click Authentication from left menu
  • Click Get Started
  • Enable Email/Password
  • Enable Google Sign-In
